# nvm

> node的版本管理工具

## 下载

https://github.com/coreybutler/nvm-windows/releases

## 常用命令

```
列出已安装的所有版本
nvm list

列出可安装的所有版本
nvm list available

安装指定版本
nvm install 14.20.0

卸载指定版本
nvm uninstall 14.20.0

使用指定版本
nvm use 14.20.0

当前使用版本
nvm current

安装最新nvm
nvm install

查看node版本
node -v

查看npm版本
npm -v
```
